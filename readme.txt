This is a text-based mastermind game in C++


1. First the game asks you if you want to activate debugging mode, which will give you the solution left corner.

2. You then need to choose between beginner, intermediary and expert, which affects the amount of letters (colored pins in the real game), the range of colors
and the number of tries you have to solve the game.

3. On each turn you get to know how many pins (letters representing colored pins) are well placed and how many are of the right color but wrongly placed
